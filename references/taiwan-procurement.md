# 台灣政府採購法制（最有利標 / 評選 / 序位法）

委外 RFP 在台灣鑲嵌於《政府採購法》及其子法。write 模式據此設計招標 / 決標 / 評選章節；review 模式據此檢查程序合法性。**法條時有修正，引用前以全國法規資料庫最新版為準。**

## 1. 招標方式三分法（差別＝廠商資格範圍）

- **公開招標** — 公告邀請所有不特定廠商；強調公開競爭。
- **選擇性招標** — 先資格審查建立合格廠商名單，再邀名單內投標。《政府採購法》第 20 條：適用經常性採購、審查耗時、需高額投標費用、資格複雜、研發等；經常性採購須建 6 家以上名單並定期檢討。
- **限制性招標** — 不經公開程序，直接邀特定廠商比價 / 議價。《政府採購法》第 22 條須符合法定條件（無廠商投標 / 獨家供應 / 緊急 / 後續擴充 / 小額採購等）。

## 2. 決標方式（決定 RFP 的評選邏輯）

- **最低標** — 符合規定且價格最低者得標。
- **評分及格最低標** — 先審查剔除表現太差者，合格者中價格最低者得標（達門檻即可，最終比價格）。
- **最有利標** — 不訂底價，綜合評選後由整體表現最好者得標（直接選「最好」）。**RFP / 建議書徵求最常綁定此方式。**

## 3. 異質採購與最有利標

- **異質採購**：標的在技術、品質、功能、效益或履約能力上顯著差異、無法僅以價格決標。認定依《政府採購法》第 52 條、施行細則第 66 條、《最有利標評選辦法》第 5 條。
- 面對異質採購 → 採最有利標，透過 RFP 廣納方案。
- **最有利標五面向**：技術 / 品質、功能 / 效益、廠商管理 / 執行能力、價格、企業社會責任（CSR）。
- **公開徵求建議書**：《政府採購法》第 34 條原則上招標前保密，但為公開徵求廠商提供參考資料則不在此限——機關得於招標前公開徵求對招標文件（規格 / 價格 / 履約條件）意見以了解市場、減少爭議；不得僅為省時直接向特定廠商詢價（洩密 / 圖利風險）。另有招標文件公開閱覽制度。

## 4. 採購評選委員會（組成不合法 → 決標可被撤銷）

- **委員 5 人以上**，具相關專門知識；**外聘專家學者不得少於 1/3**，且外聘委員不得為政府機關現職人員。
- 任務：招標前訂定 / 審定評選項目、評審標準、評定方式；辦理評選。**評選項目、子項、配分 / 權重須於招標文件明確載明，評選委員會不得變更或補充。**
- **召集人須機關首長，或由首長（或其授權人員）指定「一級主管以上」人員**；實務上「督學」「專門委員」非一級主管，不得任召集人。

## 5. 序位法（最有利標最常見評定法）

1. 個別委員對各廠商各評選項目評分後加總。
2. 依加總分數高低轉序位（最高＝序位 1）。
3. 彙整合計各廠商序位，**以平均總評分 70 分以上、序位合計值最低者為第 1 名**。
4. 優點：消弭委員間打分差異（級距化，類比會考等級制）。
5. 價格納入評分 → 總評分最高且經首長或評選委員會過半數決定者為最有利標；價格不納入 → 綜合考量總評分與價格。
6. 同序位第一處理：再行綜合評選 / 擇配分最高項目得分合計較高 / 擇獲評定序位第一較多者。

## 6. 加權計分矩陣（與國際實務一致）

- 依重要性賦予各評選項目不同權重（例：技術 40%、價格 30%）。
- 組跨部門（IT / 工程 / 終端使用者）評選團隊，採一致的數值或加權計分法維持客觀。

## 7. 程序時限與文件

- **等標期**（依招標期限標準、金額級距）：未達公告金額 ≥ 7 日；公告金額以上未達查核金額 ≥ 14 日；查核金額以上未達巨額 ≥ 21 日；巨額 ≥ 28 日。
- **投標須知**：規範投標規則、程序、資格、規格、契約條款（工程會有範本）。
- **釋疑**：廠商限期書面提疑義，機關須書面答覆並通知所有領標廠商，確保理解一致。
